ngx-relativity-date

Remarks

ngx-relativity-date is a package designed to efficiently handle date and time in angular. Built on Moment.js, this package provides a framework for a customizable figuration which provides flexibilty convenience.


Getting Started

If you are intending to use this repository you should be familiar with angular, and will need to install angular on your machine. The command needed to install angular is: npm install -g @angular/cli


Installation

To install our package you would use the node package manager. The command would be npm install ngx-relativity-date Then using the command npm start You will be able to deploy this service in localhost:4200 to see what you can do with the application.

relativityDate Property

Once ngx-relativity-date is installed, relativityDate can be accessed as a property of Date(). (See API Documentation.)


Formatting

To obtain a formatted string representation of relativityDate, use the format method with a token parameter. See the example tokens below.
Syntax Date.relativityDate.Format(tokenString);


  • Time
  • 'LT' // 10:30 AM
  • 'LTS' // 10:30:02 AM

  • Month, Day, Year
  • 'L' // 1/1/2020
  • 'l' // 01/01/2020
  • 'LL' // January 1, 2020
  • 'll' // Jan 1, 2020

  • Month, Day, Year, time
  • 'LLL' // January 1, 2020 10:30 AM
  • 'lll' // Jan 1, 2020 10:30 AM

  • Month, Day, Week, Year, time
  • 'LLLL' // Friday, January 1, 2020 10:30 AM
  • 'llll' // Fri, Jan 1, 2020 10:30 AM

The token parameter for relativityDate is compatible with all formatting of Moment.js. For more examples on date formattng please see the Moment.js documentation here.


Time Manipulation

Through the relativityDate property the value of the date object can be altered using the add() and subtract() methods. Both methods take an input of an integer, and a DatePart value.
Syntax: Date.relativityDate.add(integer, DatePart); and Date.relativityDate.subtract(integer, DatePart);
