ngx-relativity-date


Remarks

ngx-relativity-date is a package designed to efficiently handle date and time in angular. Built on Moment.js, this package provides a framework for a customizable configuration which provides developers flexibilty and convenience. ngx-relativity-date enhances features of Moment.js through object compatibility, allowing the developer to chain nultiple relativityDate functions. Conversion capabilities provide simplicity when accessing features of Moment.js and ngx-relativity-date jointly.


Getting Started

If you are intending to use this repository you should be familiar with angular, and will need to install angular on your machine. The command needed to install angular is: npm install -g @angular/cli


Installation

Use the node package manager to install ngx-relativity-date. Type the command npm install ngx-relativity-date. After the installation completes, enter the command npm start to begin using ngx-relativity-date in your application.

Formatting

To obtain a formatted string representation of relativityDate, use the format method with a token parameter. See the example tokens below.
Syntax: Date.relativityDate.format(tokenString);


  • Time
  • now.relativityDate.format('LT'); // 10:30 AM
  • now.relativityDate.format('LTS'); // 10:30:02 AM

  • Month, Day, Year
  • now.relativityDate.format('L'); // 1/1/2020
  • now.relativityDate.format('l'); // 01/01/2020

  • now.relativityDate.format('LL'); // January 1, 2020
  • now.relativityDate.format('ll'); // Jan 1, 2020

  • Month, Day, Year, time
  • now.relativityDate.format('LLL'); // January 1, 2020 10:30 AM
  • now.relativityDate.format('lll'); // Jan 1, 2020 10:30 AM

  • Month, Day, Week, Year, time
  • now.relativityDate.format('LLLL'); // Friday, January 1, 2020 10:30 AM
  • now.relativityDate.format('llll'); // Fri, Jan 1, 2020 10:30 AM

The token parameter for relativityDate is compatible with all formatting of Moment.js. For more examples please see the Moment.js documentation here.


Time Manipulation

Using the relativityDate, property the value of the date object can be altered using the add() and subtract() methods. Both methods take an input of an integer, and a DatePart value.
Syntax: Date.relativityDate.add(integer, DatePart); and Date.relativityDate.subtract(integer, DatePart);


Holidays

Holidays can customized in the default.config.ts file. The Default holidays in the ngx-relativity-date package are:

  • Christmas
  • Labor Day
  • New Year's Day
  • Martin Luther King Jr. Day
  • Washington's Birthday
  • Independence Day
  • Columbus Day
  • Veteran's Day
  • Thanksgiving Day
  • Memorial Day

Date Evaluation Methods

Dates can be evaluated using the various Date Manipulation Methods included in the ngx-relativity-date package.

  • isHoliday()
  • isBeforeDate()
  • isAfterDate()
